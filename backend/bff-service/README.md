# SmartLogix BFF Service

Este componente corresponde al Backend For Frontend del sistema SmartLogix.

## Descripción

El BFF centraliza las solicitudes del frontend y las redirige hacia los microservicios internos del sistema. Esto permite que el frontend no dependa directamente de cada microservicio y mejora la organización de la arquitectura.

## Tecnologías utilizadas

- Java 17
- Spring Boot
- Maven
- WebClient
- API REST
